# PCB Adaptateur I2C Level Shifter

Carte PCB sur mesure conçue sous **KiCad 8** permettant la conversion bidirectionnelle d'un bus I2C entre deux niveaux de tension (**3,3 V ↔ 5 V**), format breadboard.

## Caractéristiques

- Conversion bidirectionnelle I2C (SDA + SCL)
- Compatible 3,3 V ↔ 5 V
- Format adapté breadboard
- Conception schématique + routage PCB complets
